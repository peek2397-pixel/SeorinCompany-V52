서린 물류 포털 V7 - B2C KPI 전체 내용 통합

이번 버전에서는 기존에 올려주신 B2C KPI 프로그램 전체를
포털의 'KPI 관리' 메뉴 안에 실제로 넣었습니다.

포함된 KPI 기능
- 대시보드
- 개인 일일수량
- 지원분배
- 공동작업
- 월별평가
- 기간 종합평가
- 직원관리
- 평가기준
- 엑셀/CSV/백업

적용 방법
1. 이 압축파일을 새 폴더에 풉니다.
2. index.html을 실행합니다.
3. Ctrl+F5로 새로고침합니다.
4. KPI 관리 메뉴를 클릭합니다.

중요
- 현재 KPI 데이터는 기존 프로그램 방식대로 해당 브라우저의 localStorage에 저장됩니다.
- 즉, 다른 PC나 다른 직원의 휴대폰과 자동 공유되지는 않습니다.
- 다음 단계에서 KPI 데이터를 Supabase에 저장하도록 바꾸면 여러 사용자가 같은 데이터를 공유할 수 있습니다.
- 일반 직원에게 KPI 메뉴 권한을 주지 않으면 화면 자체가 보이지 않습니다.
