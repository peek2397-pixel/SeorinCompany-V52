서린 물류 포털 V5 - 회사 사원번호 그대로 가입

변경 내용
- EMP001 형식을 강제하지 않습니다.
- 회사에서 실제 사용하는 사원번호를 그대로 입력합니다.
- 숫자만 있는 사원번호, 영문 포함 사원번호, 하이픈 포함 사원번호도 사용할 수 있습니다.
- 내부적으로만 유효한 이메일 주소로 변환하여 Supabase에 저장합니다.
- 기존 EMP001 관리자 계정도 계속 로그인됩니다.

적용 방법
1. 압축을 풉니다.
2. 기존 포털 폴더의 app.js와 index.html을 새 파일로 교체합니다.
3. config.js는 현재 정상 연결된 파일을 그대로 사용합니다.
4. 홈페이지에서 Ctrl+F5를 누릅니다.
5. 회원가입에서 회사 사원번호를 그대로 입력합니다.

예:
- 10027
- 2026-015
- A1024
