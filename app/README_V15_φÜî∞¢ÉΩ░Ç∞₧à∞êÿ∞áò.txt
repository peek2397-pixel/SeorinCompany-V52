서린 물류 포털 V15 - 회원가입 이메일 오류 수정본

수정 내용
- 직원은 계속 회사 사원번호와 비밀번호만 사용합니다.
- 화면에는 이메일이 표시되지 않습니다.
- 신규 가입 시 내부적으로만 다음 형식을 사용합니다.
  seorin.portal.사원번호@gmail.com
- 기존 EMP001 관리자와 이전 방식으로 가입된 계정도 계속 로그인되도록
  예전 내부 주소를 함께 확인합니다.

적용 방법
1. 이 압축을 새 폴더에 풉니다.
2. index.html을 실행합니다.
3. Ctrl + F5를 누릅니다.
4. 회원가입을 다시 시도합니다.

김헌정 가입 예시
- 사원번호: 202305022
- 이름: 김헌정
- 비밀번호: 6자 이상

Supabase 설정
- Authentication → Allow new users to sign up: ON
- Confirm email: OFF
- Email provider: ON

추가 SQL은 필요하지 않습니다.
단, V14_사원번호전용로그인_확인.sql을 아직 실행하지 않았다면 먼저 실행하세요.
