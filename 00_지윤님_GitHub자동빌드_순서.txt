서린컴퍼니 V41 — GitHub 자동 빌드 사용법

이 ZIP은 지윤님 PC에서 npm 빌드를 하지 않고,
GitHub의 Windows 서버에서 실제 Setup.exe를 자동 생성하도록 만든 프로젝트입니다.

────────────────────────
1. GitHub 저장소 만들기
────────────────────────
1) github.com에 로그인
2) 오른쪽 위 + 버튼 → New repository
3) Repository name: SeorinCompany-V41
4) Private 선택
5) Create repository 클릭

────────────────────────
2. 파일 올리기
────────────────────────
1) 이 ZIP을 완전히 압축 해제
2) GitHub 저장소에서 “uploading an existing file” 클릭
3) 압축 해제한 폴더 안의 모든 파일과 폴더를 업로드
   - .github 폴더가 반드시 포함되어야 합니다.
   - app, build, SQL 폴더도 포함되어야 합니다.
4) Commit changes 클릭

주의:
GitHub 웹 업로드에서 .github 폴더가 안 보이면,
Windows 탐색기의 보기 → 숨긴 항목을 켜세요.

────────────────────────
3. Setup.exe 자동 만들기
────────────────────────
1) 저장소 위쪽 Actions 클릭
2) 왼쪽 “Build SeorinCompany Windows Setup” 클릭
3) 오른쪽 Run workflow 클릭
4) 다시 초록색 Run workflow 클릭
5) 약 5~10분 기다림
6) 완료된 실행 내역 클릭
7) 화면 아래 Artifacts에서
   “SeorinCompany_Setup_Windows” 다운로드
8) ZIP을 풀면 실제 설치파일이 나옵니다.

생성 파일:
SeorinCompany_Setup_1.0.0.exe

────────────────────────
설치파일 기능
────────────────────────
- 설치 마법사
- 설치 경로 선택
- 바탕화면 독도로션 아이콘 자동 생성
- 시작 메뉴 등록
- 설치 완료 후 자동 실행
- 프로그램 제거 지원
- 기존 Supabase 데이터 공동 연동
- V41 1:1·그룹 메신저 포함

────────────────────────
메신저 SQL
────────────────────────
SQL/V41_직원선택형_메신저.sql을
Supabase SQL Editor에서 회사 전체에 한 번 실행하세요.
